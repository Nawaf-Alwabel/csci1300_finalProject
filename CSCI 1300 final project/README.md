# A cool town

## Theme

[in medieval times. It will be centered around a village that is currently during an election for a new mayor. Your best friend wants to run for mayor, but they are a bit of a "joke" around the village.]

## Goal

[Gain 200 gold to bribe everyone and make your friend "Mark" win the election]

## How to Compile

copy and paste this in VS terminal (ALL FILES MUST BE IN THE FOLDER, INCLUDING TXT FILE)

g++ -std=c++17 -Wall -Werror -Wpedantic main.cpp Item.cpp Character.cpp Fisher.cpp Hunter.cpp Player.cpp Game.cpp -o game

## How to Run

1- download the folder (ignore all scattered files, they are old)
2- import/copy it into VS code
3- make sure all files are in the folder
4- run the compile above
5- open the folder and run "game"


## How to Play

figuring it out is apart of the game! just don't lose all your money ;) 


## Roles

fisher starts with 20 gold.

hunter starts with 20 gold.

gambler starts with 50 gold and can enter the casino on the first day.

the casino requires at least 50 gold to enter.
